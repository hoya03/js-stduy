// 🎈 배열 + const 🎈 const를 사용하여 배열을 만들때는 안에 내용을 변경할수없다 (기본적인 상황에서는) const 값이 변할때는 인테리어 업자들이 집내부를 바꾸어도 집 외부는 바뀌지 않는것과 같다! 👌 // 🎈 NESTED arrays 다차원 배열 🎈 const board = [ ['O', null, 'X'], [null, 'X', 'O'], ['X', 'O', null], ] board (3) [Array(3), Array(3), Array(3)] 0: (3) ['O', null, 'X'] 1: (3) [null, 'X', 'O'] 2: (3) ['X', 'O', null] length: 3 [[Prototype]]: Array(0) 🎄board[1] // 이렇게 변수명[1]<= 인덱스 번호를 입력하면 원하는 자료에 접근이 가능하다. (3) [null, 'X', 'O'] board[1][0] 변수명[1][0]<= 1번 인덱스의 자료를 불러온뒤 뒤에 다시 인덱스 숫자를 null 입력을하면 원하는 자리수의 자료를 불러올수있다. board[1][2] 'O' 🎄만약 중첩배열의 내용을 바꾸고 싶다면? board[1][2] = 'X' 입력을하게된다면 결과값음 board[1] (3) [null, 'X', 'X'] 이렇게 변경이 가능하다! 이렇게 중첩배열을 통해서 게임보드판을 만들수있다.
